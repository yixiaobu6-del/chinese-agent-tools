# 中文 Agent 工具链

[![HTML](https://img.shields.io/badge/HTML-standalone-orange)]()
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-blue)]()

中文 AI Agent 工具链展示与组合可视化工具。

## 功能

- **工具展示**：各 Agent 工具功能概述
- **组合可视化**：4 种组合场景的节点流程图
- **场景推荐**：按需求推荐工具组合
- **交互式探索**：点击节点查看详情

## 使用

打开 `tool_showcase.html` 即可使用。
